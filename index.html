Hawk PEAR channel

Registering the channel: 
pear channel-discover clockwerx.github.com/hawk
Listing available packages: 
pear remote-list -c hawk
Installing a package: 
pear install hawk/package_name
Installing a specific version/stability: 
pear install hawk/package_name-1.0.0
pear install hawk/package_name-beta
Receiving updates via a feed: 
https://hawk.htw.com.au/feed.xml
